If you are planning to create a blog using Next.js I think you should give a try to [Outstatic. ](https://outstatic.com)Outstatic is an open source content management system for static websites using [**Next.js**](https://nextjs.org/).

The setup is pretty straightforward, the editor is amazing and you don't need to care about creating an account in another service. You just need an Github account and a Oauth app configured.
